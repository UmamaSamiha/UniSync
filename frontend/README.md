# UniSync - Peer Tutoring Frontend

This is the frontend interface for the Peer Tutoring module.

### Technologies Used
* React.js
* Axios (for API communication)
* HTML/CSS

### Features
## Features

 **Feature 1: Student Request Creation** - The system allows students to create tutoring requests for a specific subject (e.g., CSE470, MAT215).

**Feature 2: Tutor Browsing & Discovery** - A dynamic list of available tutors is provided for students to browse, complete with bios and ratings.

**Feature 3: Peer-to-Peer Sending** - The system allows students to send tutoring requests directly to other students who are registered as tutors.

**Feature 4: Request Management** - Tutors can view incoming requests on their dashboard and choose to either accept or reject them. these are my features.
